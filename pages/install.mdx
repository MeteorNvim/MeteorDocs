# Installation

**To get started with MeteorNvim, follow these steps**

**but first make sure you have the `xsel` dependency installed**

## 1. Clone the Repository

```bash
git clone https://github.com/MeteorNvim/MeteorNvim.git
```

## 2. Warning: Backup Your Existing Configuration

Before proceeding, it's crucial to backup your existing Neovim configuration if you have one. This ensures you can revert to your previous setup if needed.

```bash
mv ~/.config/nvim ~/.config/nvim_backup
```

## 3. Move to ~/.config/nvim

```bash
cd MeteorNvim
cp -r * ~/.config/nvim
```

## 4. Final Steps

Now that you've set up MeteorNvim, launch Neovim and let the magic unfold!
