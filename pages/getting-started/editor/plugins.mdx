## How to add plugins
Go to `~/.config/nvim/lua/userconfig/`
and put the plugin inside the `Plugins` table

```lua filename="plugins.lua" {3-8} copy
local Plugins = {
  -- DiscordRPC
  {
   'MeteorNvim/presence.nvim',
    config = function ()
      require("userconfig.config.presence")
    end
  },
}

return Plugins
```
