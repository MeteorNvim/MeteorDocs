## Add Mappings

You can modify or add your own mappings in `~/.config/nvim/lua/userconfig/mappings.lua`

**Example**
```lua filename="mappings.lua" {} copy
M.map('n', "<C-t>", "<CMD>:terminal<CR>", { silent = true })
```
